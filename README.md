# Personal Blog Server Side

using koa2  
using async/await instead of Promise  
using koa-router@next  
using a self-made, decent log module  
based on mongodb  
seperated config file  
using ORM

node > 7.0 required  
npm install  
change config.example.js name to config.js  

## run
npm run run
